## Packages
framer-motion | For smooth piece movement and UI animations
lucide-react | Icons for the game interface
clsx | Utility for conditional classes
tailwind-merge | Utility for merging tailwind classes

## Notes
The game logic is complex and primarily client-side state driven, but syncs with the backend for save/load.
The board layout relies on CSS Grid (11x11).
Theme is "Retro Board Game" - dark green felt, clean cards.
